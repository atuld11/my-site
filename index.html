How it works

One message to log.
One message to settle.

01

Add the bot

Add SplitKaro to any WhatsApp group like you'd add a contact. It joins as a participant and starts tracking immediately. Nothing changes about the group.

02

Type what you paid

"Paid 2000 for Airbnb." The bot reads it, calculates the split, and posts the breakdown. No form. No tap sequence. No new interface to learn.

03

Settle with one message

"Paid Rahul 500." The bot records it, updates every balance, and confirms the settlement. Ask "who owes what" anytime — you get a clean answer.

SplitKaro only reads messages that include a spending keyword. No other messages are read or stored. Your conversations stay yours.

Why this exists
Every trip I've taken with friends ends the same way. Someone pulls up a notes app. Someone else starts a WhatsApp thread. Someone who paid for the hotel is still waiting three weeks later.

We build software that automates how large buildings run — cutting energy waste, making operations smarter, more sustainable. We're talking ₹500 crore+ in infrastructure managed. But splitting a Goa trip with six friends? Still a mess.

The problem isn't math. It's friction. Every solution needs a new app, a new login, and a ten-minute group chat to convince people to use it. Nobody does that. Everyone's already on WhatsApp.

So we built SplitKaro to live where the conversation already happens.
